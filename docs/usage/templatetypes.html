Template Types

<

There are two basic templates in GWSDroidLava. One is the clearsilver template with the extension of cs. The other template is the javadoc template with the extension of jd.

ClearSilver

The ClearSilver template features are provided by the JSivler jar included in GWSDroidLava. Basic full 100% compatibility with the ClearSivler macros and calls. Those syntaxes and command page resources are:

  • template commands
  • Expressions
  • Macros
  • functions
JSilver provides a further inline directive syntax explained in the middle of this page.

Javadoc

The other template type is javadoc. In the javadoc template only page properties at the top and javadoc commands such as docroot are processed. One of the main properties you often put at the top is the command telling the GWSDroidlava doclet that the body of content follows that clearsilver needs to act upon in rendering the template to html.
